matches?(string, regex) click to toggle source

Public: Checks if a string contains a specific regular expression and returns the match, if there is any.

  • string - String which should be checked.

  • regex - Regular expression to search for in string.

Example

regex_test = RegexTest.new
regex_test.matches? 'This is a test.', /i.*/

# => "is is a test."

Returns the match, if any.

# File regextest.rb, line 37
def matches? string, regex
  regex.match string
end
